# Build a Digital Mood Board App

Pinterest is great but sometimes you just want a blank canvas where you can throw images, colors, text, and vibes together without an algorithm telling you what to look at. Let's build your own mood board app.

## Why Would You Build This?

Real talk ‚Äî mood boards aren't just for aesthetics (although they're great for that). People use them for:

- **Planning a rebrand** ‚Äî collecting colors, fonts, and visual inspo before redesigning a website or social media
- **Apartment decorating** ‚Äî pulling together furniture, paint colors, and textures before you commit
- **Event planning** ‚Äî weddings, parties, photoshoots ‚Äî visualizing the vibe before spending money
- **Creative projects** ‚Äî album art, YouTube thumbnails, merch designs
- **Vision boards** ‚Äî goals, affirmations, dream life visualization

And building it yourself means no watermarks, no subscription fees, and no ads for things you already bought.

<Callout variant="denise-says">
I use mood boards for literally everything ‚Äî redesigning my website, planning content, even figuring out what I want my apartment to look like. Building one as a coding project means you learn drag-and-drop, canvas layouts, and image handling, all while making something you'll genuinely use.
</Callout>

## What We're Building

- A blank canvas you can add items to
- Drag and drop to rearrange
- Add images (from URL or upload)
- Add color swatches
- Add text notes
- Save your board to come back to later

## Step 2: Draggable Item Component

<CodeBlock
  code="import { useState } from 'react';

function BoardItem({ item, onMove, onDelete }) {
  const [isDragging, setIsDragging] = useState(false);
  const [dragStart, setDragStart] = useState({ x: 0, y: 0 });

  const handleMouseDown = (e) => {
    setIsDragging(true);
    setDragStart({
      x: e.clientX - item.x,
      y: e.clientY - item.y,
    });
  };

  const handleMouseMove = (e) => {
    if (!isDragging) return;
    onMove(item.id, {
      x: e.clientX - dragStart.x,
      y: e.clientY - dragStart.y,
    });
  };

  const handleMouseUp = () => {
    setIsDragging(false);
  };

  return (
    <div
      className={`board-item ${isDragging ? 'dragging' : ''}`}
      style={{
        left: item.x,
        top: item.y,
        width: item.width,
        transform: 'rotate(' + item.rotation + 'deg)',
      }}
      onMouseDown={handleMouseDown}
      onMouseMove={handleMouseMove}
      onMouseUp={handleMouseUp}
    >
      {item.type === 'image' && (
        <img src={item.content} alt='' draggable={false} />
      )}
      {item.type === 'color' && (
        <div className='color-swatch' style={{ background: item.content }} />
      )}
      {item.type === 'text' && (
        <p className='board-text'>{item.content}</p>
      )}

      <button className='delete-item' onClick={() => onDelete(item.id)}>x</button>
    </div>
  );
}"
  language="jsx"
  explanation="This component makes items draggable. When you click and hold (mouseDown), it starts tracking your mouse position. As you move (mouseMove), it updates the item's position. When you let go (mouseUp), it stops. The rotation gives each item a slightly tilted, collage-style look. Each item type renders differently ‚Äî images show the photo, colors show a swatch, text shows the note."
/>

## Step 4: The Full Board

<CodeBlock
  code="function MoodBoard() {
  const [items, setItems] = useState([]);
  const [boardName, setBoardName] = useState('My Mood Board');

  const addItem = (itemData) => {
    setItems(prev => [...prev, {
      ...itemData,
      id: Date.now(),
      x: 100 + Math.random() * 200,
      y: 100 + Math.random() * 200,
    }]);
  };

  const moveItem = (id, position) => {
    setItems(prev => prev.map(item =>
      item.id === id ? { ...item, ...position } : item
    ));
  };

  const deleteItem = (id) => {
    setItems(prev => prev.filter(item => item.id !== id));
  };

  const saveBoard = () => {
    const data = { name: boardName, items };
    localStorage.setItem('moodboard', JSON.stringify(data));
  };

  return (
    <div className='mood-board-app'>
      <header className='board-header'>
        <input
          value={boardName}
          onChange={e => setBoardName(e.target.value)}
          className='board-name-input'
        />
        <button onClick={saveBoard} className='save-btn'>
          Save Board
        </button>
      </header>

      <Toolbar onAddItem={addItem} />

      <div className='canvas'>
        {items.map(item => (
          <BoardItem
            key={item.id}
            item={item}
            onMove={moveItem}
            onDelete={deleteItem}
          />
        ))}
        {items.length === 0 && (
          <p className='empty-canvas'>
            Click the buttons above to start adding to your board ‚ú®
          </p>
        )}
      </div>
    </div>
  );
}"
  language="jsx"
  explanation="The main component manages all the items on the board. New items get placed at a slightly random position so they don't all stack on top of each other. The board name is editable (click on it and type). Save Board stores everything in localStorage so you can close the tab and come back to it."
/>

## Level Up Ideas

- **Multiple boards** ‚Äî save and switch between different mood boards
- **File upload** ‚Äî drag images from your computer instead of pasting URLs
- **Resize handles** ‚Äî drag corners to resize items
- **Layers** ‚Äî bring items forward or send them backward
- **Export as image** ‚Äî save your board as a PNG to share
- **Collaborative** ‚Äî share a board link and build together in real-time
