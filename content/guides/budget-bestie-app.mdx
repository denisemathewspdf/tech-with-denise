# Build a Budget Bestie App

Budgeting apps are either way too complicated or way too ugly. Let's build one that's simple, cute, and actually helpful â€” like a bestie who's good with money.

## What We're Building

- Add expenses with categories (food, shopping, coffee, going out, subscriptions)
- See a visual breakdown of where your money goes
- Set monthly budgets per category
- Get a vibe check on your spending (are you on track or nah?)

<Callout variant="denise-says">
I built my first budget app because I needed to understand where my money kept disappearing. Turns out it was Uber Eats. The app I built to track it was also how I learned about arrays, reduce(), and localStorage. Win-win (except for Uber Eats).
</Callout>

## Step 2: Add Expense Form

<CodeBlock
  code="import { useState } from 'react';

function AddExpense({ onAdd }) {
  const [amount, setAmount] = useState('');
  const [category, setCategory] = useState('food');
  const [note, setNote] = useState('');

  const handleSubmit = (e) => {
    e.preventDefault();
    if (!amount) return;

    onAdd({
      id: Date.now(),
      amount: parseFloat(amount),
      category,
      note,
      date: new Date().toISOString().split('T')[0],
    });

    // Reset form
    setAmount('');
    setNote('');
  };

  return (
    <form onSubmit={handleSubmit} className='add-expense'>
      <h3>Add Expense</h3>

      {/* Amount input */}
      <div className='amount-input'>
        <span className='dollar-sign'>$</span>
        <input
          type='number'
          step='0.01'
          placeholder='0.00'
          value={amount}
          onChange={e => setAmount(e.target.value)}
        />
      </div>

      {/* Category picker â€” emoji buttons */}
      <div className='category-grid'>
        {categories.map(cat => (
          <button
            key={cat.id}
            type='button'
            onClick={() => setCategory(cat.id)}
            className={`cat-btn ${category === cat.id ? 'selected' : ''}`}
            style={{ borderColor: category === cat.id ? cat.color : 'transparent' }}
          >
            <span>{cat.emoji}</span>
            <span className='cat-label'>{cat.name}</span>
          </button>
        ))}
      </div>

      {/* Note */}
      <input
        type='text'
        placeholder='What was it for? (optional)'
        value={note}
        onChange={e => setNote(e.target.value)}
        className='note-input'
      />

      <button type='submit' className='submit-btn'>
        Add Expense
      </button>
    </form>
  );
}"
  language="jsx"
  explanation="The form has three parts: a big amount input with a dollar sign, emoji category buttons (way more fun than a dropdown), and an optional note. When submitted, it creates an expense object with a unique ID (using Date.now()), parses the amount as a number, and calls onAdd to save it. Then it resets so you can add another one."

## Step 3: Spending Summary

<CodeBlock
  code="function SpendingSummary({ expenses, month }) {
  // Total spent this month
  const total = expenses.reduce((sum, e) => sum + e.amount, 0);

  // Spending by category
  const byCategory = categories.map(cat => {
    const catExpenses = expenses.filter(e => e.category === cat.id);
    const catTotal = catExpenses.reduce((sum, e) => sum + e.amount, 0);
    const percentage = total > 0 ? (catTotal / total) * 100 : 0;

    return {
      ...cat,
      total: catTotal,
      percentage: Math.round(percentage),
      count: catExpenses.length,
    };
  }).filter(cat => cat.total > 0); // Only show categories with spending

  return (
    <div className='summary'>
      <h3>This Month</h3>
      <p className='total-amount'>${total.toFixed(2)}</p>

      {/* Visual bar breakdown */}
      <div className='spending-bar'>
        {byCategory.map(cat => (
          <div
            key={cat.id}
            className='bar-segment'
            style={{
              width: cat.percentage + '%',
              backgroundColor: cat.color,
            }}
            title={cat.name + ': $' + cat.total.toFixed(2)}
          />
        ))}
      </div>

      {/* Category breakdown list */}
      <div className='category-list'>
        {byCategory
          .sort((a, b) => b.total - a.total)
          .map(cat => (
            <div key={cat.id} className='category-row'>
              <span className='cat-dot' style={{ background: cat.color }} />
              <span className='cat-name'>{cat.emoji} {cat.name}</span>
              <span className='cat-amount'>${cat.total.toFixed(2)}</span>
              <span className='cat-percent'>{cat.percentage}%</span>
            </div>
          ))
        }
      </div>
    </div>
  );
}"
  language="jsx"
  explanation="This component does the math on your spending. reduce() adds up all the amounts to get your total. Then it groups expenses by category and calculates what percentage each category is. The colored bar at the top gives you an instant visual â€” if the coffee segment is suspiciously large, you know why. Categories are sorted by most spending first."
/>

## Step 4: Vibe Check (Budget Status)

<CodeBlock
  code="function VibeCheck({ totalSpent, monthlyBudget }) {
  const percentage = (totalSpent / monthlyBudget) * 100;
  const remaining = monthlyBudget - totalSpent;
  const daysLeft = getDaysLeftInMonth();
  const dailyBudget = remaining > 0 ? remaining / daysLeft : 0;

  // Determine the vibe
  let vibe;
  if (percentage < 50) {
    vibe = { emoji: 'âœ¨', message: 'Looking great! You are well under budget.', color: '#A8D8EA' };

  );
}

function getDaysLeftInMonth() {
  const now = new Date();
  const lastDay = new Date(now.getFullYear(), now.getMonth() + 1, 0);
  return lastDay.getDate() - now.getDate();
}"

## Level Up Ideas

- **Monthly comparison** â€” see if you're spending more or less than last month
- **Savings goals** â€” set goals ("concert tickets: $200") and track progress
- **Receipt scanner** â€” use your phone camera to auto-fill expense amounts
- **Recurring expenses** â€” auto-add subscriptions on the 1st of every month
- **Export to CSV** â€” download your data for spreadsheet nerds
- **Split expenses** â€” track shared costs with friends and who owes who
