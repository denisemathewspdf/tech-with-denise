# Organize Your Messy Desktop with Python

Be honest. How many random files are on your desktop right now? Screenshots from three months ago? That PDF you downloaded once? A folder called "New Folder (3)"? Let's write a Python script that cleans that mess up automatically.

## What We're Building

A script that:
- Scans your desktop (or any folder)
- Sorts files into organized folders by type (Images, Documents, Music, etc.)
- Runs in seconds
- Can be scheduled to run automatically

## Step 3: The Sorting Function

<CodeBlock
  code="def get_category(filename):
    '''Figure out which folder a file belongs in based on its extension.'''
    extension = os.path.splitext(filename)[1].lower()

    for category, extensions in FILE_CATEGORIES.items():
        if extension in extensions:
            return category

    return 'Other'  # Files that don't match any category


def organize_folder(folder_path):
    '''Sort all files in a folder into categorized subfolders.'''

    # Count how many files we moved
    moved = 0

    for filename in os.listdir(folder_path):
        file_path = os.path.join(folder_path, filename)

        # Skip folders ‚Äî we only want to sort files
        if os.path.isdir(file_path):
            continue

        # Skip hidden files (like .DS_Store on Mac)
        if filename.startswith('.'):
            continue

        # Figure out where this file goes
        category = get_category(filename)

        # Create the category folder if it doesn't exist
        category_path = os.path.join(folder_path, category)
        if not os.path.exists(category_path):
            os.makedirs(category_path)
            print(f'  Created folder: {category}/')

        # Move the file!
        destination = os.path.join(category_path, filename)
        shutil.move(file_path, destination)
        print(f'  Moved: {filename} -> {category}/')
        moved += 1

    return moved"
  language="python"
  explanation="The organize_folder function goes through every file in the folder. For each file, it checks the extension to figure out the category, creates the category folder if needed, and moves the file. It skips folders and hidden files so it doesn't mess up anything important. The print statements show you exactly what's happening as it runs."
/>

## Step 5: Add an Undo Feature

Because sometimes you want your mess back:

<CodeBlock
  code={"import json\nfrom datetime import datetime\n\ndef organize_folder_with_log(folder_path):\n    '''Same as organize_folder, but saves a log so you can undo.'''\n    log = []\n    moved = 0\n\n    for filename in os.listdir(folder_path):\n        file_path = os.path.join(folder_path, filename)\n\n        if os.path.isdir(file_path) or filename.startswith('.'):\n            continue\n\n        category = get_category(filename)\n        category_path = os.path.join(folder_path, category)\n\n        if not os.path.exists(category_path):\n            os.makedirs(category_path)\n\n        destination = os.path.join(category_path, filename)\n        shutil.move(file_path, destination)\n\n        log.append({\n            'file': filename,\n            'from': file_path,\n            'to': destination\n        })\n        moved += 1\n\n    # Save the log so we can undo later\n    log_path = os.path.join(folder_path, '.organize_log.json')\n    with open(log_path, 'w') as f:\n        json.dump(log, f, indent=2)\n\n    return moved\n\n\ndef undo_organize(folder_path):\n    '''Move everything back to where it was.'''\n    log_path = os.path.join(folder_path, '.organize_log.json')\n\n    if not os.path.exists(log_path):\n        print('No log found. Nothing to undo.')\n        return\n\n    with open(log_path, 'r') as f:\n        log = json.load(f)\n\n    for entry in log:\n        if os.path.exists(entry['to']):\n            shutil.move(entry['to'], entry['from'])\n            print('  Restored: ' + entry['file'])\n\n    os.remove(log_path)\n    print('Undo complete!')"}
  language="python"
  explanation="The organize function now saves a log file (.organize_log.json) that remembers where every file came from. The undo function reads that log and moves everything back. It's like Ctrl+Z for your file system. The log file is hidden (starts with a dot) so it won't clutter your folder."
/>

## Level Up Ideas

- **Schedule it** ‚Äî use cron (Mac/Linux) or Task Scheduler (Windows) to run it daily
- **Sort by date** ‚Äî create subfolders like Images/2025-04/ based on when files were created
- **Smart rename** ‚Äî rename screenshots from "IMG_4521.png" to "screenshot-2025-04-20.png"
- **Desktop wallpaper** ‚Äî randomly set your wallpaper from your Images folder
- **Size report** ‚Äî show how much space each category takes up
