# Build a Self-Care Tracker App

You've downloaded a million self-care apps. They're either ugly, full of ads, or track things you don't care about. What if you built your own? One that tracks exactly what matters to YOU, looks cute, and doesn't cost $9.99/month?

<Callout variant="tip">

## What We're Building

A daily tracker for:
- Water intake (glasses per day)
- Sleep hours
- Mood (emoji-based, obviously)
- Skincare routine (morning/night checklist)
- Movement (did you move your body today?)
- A daily affirmation or journal note

<Callout variant="denise-says">
This app taught me localStorage, date handling, and state management ‚Äî all because I wanted to know if I was drinking enough water. Learning is so much easier when you're building something for yourself.
</Callout>

## Step 5: Save Data with localStorage

<CodeBlock
  code="// Save today's data to localStorage (persists between sessions)
function saveDay(date, data) {
  const allData = JSON.parse(localStorage.getItem('selfcare') || '{}');
  allData[date] = data;
  localStorage.setItem('selfcare', JSON.stringify(allData));
}

// Load today's data
function loadDay(date) {
  const allData = JSON.parse(localStorage.getItem('selfcare') || '{}');
  return allData[date] || {
    water: 0,
    sleep: 0,
    mood: null,
    skincare: { morning: false, night: false },
    movement: false,
    note: '',
  };
}

// Get the last 7 days for a weekly view
function getWeekData() {
  const days = [];
  for (let i = 6; i >= 0; i--) {
    const date = new Date();
    date.setDate(date.getDate() - i);
    const dateStr = date.toISOString().split('T')[0];
    days.push({
      date: dateStr,
      ...loadDay(dateStr),
    });
  }
  return days;
}"
  language="javascript"
  explanation="localStorage lets you save data in the browser that persists even when you close the tab. saveDay stores a day's data under its date key. loadDay retrieves it (or returns defaults if it's a new day). getWeekData grabs the last 7 days so you can see your trends. No database needed ‚Äî it all lives in your browser."
/>

## Step 6: Putting It All Together

<CodeBlock
  code="import { useState, useEffect } from 'react';

function SelfCareApp() {
  const today = new Date().toISOString().split('T')[0];
  const [data, setData] = useState(loadDay(today));

  // Auto-save whenever data changes
  useEffect(() => {
    saveDay(today, data);
  }, [data, today]);

  const updateField = (field, value) => {
    setData(prev => ({ ...prev, [field]: value }));
  };

  return (
    <div className='self-care-app'>
      <header>
        <h1>Daily Check-in</h1>
        <p className='date'>{new Date().toLocaleDateString('en-US', {
          weekday: 'long',
          month: 'long',
          day: 'numeric'
        })}</p>
      </header>

      <MoodPicker
        currentMood={data.mood}
        onSelect={mood => updateField('mood', mood)}
      />

      <WaterTracker
        glasses={data.water}
        onUpdate={count => updateField('water', count)}
      />

      <SkincareTracker
        skincare={data.skincare}
        onToggle={routine => updateField('skincare', {
          ...data.skincare,
          [routine]: !data.skincare[routine]
        })}
      />

      {/* Movement toggle */}
      <div className='tracker-card'>
        <h3>Movement</h3>
        <button
          onClick={() => updateField('movement', !data.movement)}
          className={`movement-btn ${data.movement ? 'active' : ''}`}
        >
          {data.movement ? 'Yes! I moved today üí™' : 'Did you move today?'}
        </button>
      </div>

      {/* Daily note */}
      <div className='tracker-card'>
        <h3>Daily Note</h3>
        <textarea
          value={data.note}
          onChange={e => updateField('note', e.target.value)}
          placeholder='How was your day? Any wins? Anything on your mind?'
          rows={3}
        />
      </div>
    </div>
  );
}"
  language="jsx"
  explanation="This is the main app component that ties everything together. Each tracker component gets its current data and a function to update it. The useEffect auto-saves to localStorage whenever anything changes, so you never lose your data. The date shows at the top so you know what day you're tracking."
/>
