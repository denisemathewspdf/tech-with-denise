# Build Your Dream Portfolio Website

Every tutorial portfolio site looks the same — dark theme, generic "Hi, I'm a developer" text, the same three project cards. Boring. Let's build one that actually has personality. YOUR personality.

## What We're Building

A portfolio that has:
- A hero section with your name and vibe
- An about section that doesn't read like a resume
- A projects grid that shows off your work
- A contact section that's not just a sad email link
- Your own color palette and fonts (not Bootstrap defaults)

<Callout variant="denise-says">
Your portfolio is literally your first impression. It should make someone go "oh, I like this person" not "oh, another developer template." I'm going to show you how to build one that stands out, and you don't need React or any fancy framework — just HTML, CSS, and a tiny bit of JavaScript.
</Callout>

## Step 4: Smooth Scroll + Animation

Add a tiny bit of JavaScript for polish:

<CodeBlock
  code={"// Smooth scroll for navigation links\ndocument.querySelectorAll('a[href^=\"#\"]').forEach(link => {\n  link.addEventListener('click', e => {\n    e.preventDefault();\n    const target = document.querySelector(link.getAttribute('href'));\n    if (target) target.scrollIntoView({ behavior: 'smooth' });\n  });\n});\n\n// Fade-in animation on scroll\nconst observer = new IntersectionObserver(entries => {\n  entries.forEach(entry => {\n    if (entry.isIntersecting) {\n      entry.target.classList.add('visible');\n    }\n  });\n}, { threshold: 0.1 });\n\ndocument.querySelectorAll('section').forEach(section => {\n  section.classList.add('fade-in');\n  observer.observe(section);\n});"}
  language="javascript"
  explanation="The first part makes anchor links scroll smoothly instead of jumping. The second part adds a fade-in animation when sections come into view as you scroll down. It uses IntersectionObserver — which watches for elements entering the viewport. Add the CSS below to make the fade-in work."
/>

<CodeBlock
  code="/* Add this to your CSS for the fade-in effect */
.fade-in {
  opacity: 0;
  transform: translateY(20px);
  transition: opacity 0.6s ease, transform 0.6s ease;
}

.fade-in.visible {
  opacity: 1;
  transform: translateY(0);
}"
  language="css"
  explanation="Sections start invisible and slightly below their final position. When they scroll into view, the JavaScript adds the 'visible' class, which triggers the smooth fade-up animation. This small detail makes your portfolio feel so much more polished."
/>

## Step 5: Deploy It for Free

Put it online so people can actually see it:

1. Push your code to **GitHub**
2. Go to your repo settings
3. Scroll to **GitHub Pages**
4. Select your branch and save
5. Your site is live at `yourusername.github.io/portfolio`

Or use **Vercel** or **Netlify** for even easier deployment — just drag and drop your folder.
